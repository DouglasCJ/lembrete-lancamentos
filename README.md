## 📖 Sobre o Projeto

O **Lembrete de Lançamentos** foi desenvolvido para auxiliar a equipe de lançamentos de cartão de uma instituição financeira. O principal problema que ele resolve é a gestão e o acompanhamento de lançamentos pendentes, garantindo que a equipe seja lembrada de realizar as atividades nos prazos corretos, mitigando erros por esquecimento e melhorando o fluxo de trabalho diário.

Trata-se de uma aplicação rápida, dinâmica e totalmente focada em produtividade.

---

## ✨ Funcionalidades

- 📥 **Importação de Planilhas:** Facilidade na inserção de dados em massa através de arquivos `.xlsx` ou `.xls`.
- 👀 **Visualização de Lançamentos:** Acompanhamento dos lançamentos separados por "Pendentes" e "Concluídos".
- 🔄 **Sincronização na Nuvem:** Salvamento e sincronização em tempo real utilizando Supabase.
- ✅ **Ações Individuais e em Massa:** Permite concluir tarefas uma a uma ou concluir várias tarefas simultaneamente.
- 🔍 **Filtros Avançados:** Filtro por competência, responsável e status do lançamento.
- 📊 **Resumo Estatístico:** Cards interativos contendo totais de lançamentos aguardando e já concluídos.
- 📩 **Notificação via E-mail:** Integração para o envio de e-mails de lembretes e alertas de conclusão *(via Resend)*.
- 📤 **Exportação de Relatórios:** Possibilidade de baixar dados filtrados de volta para formato Excel.

   *(Lembre-se de substituir o link pelo caminho correto do seu repositório)*

   ```bash

   ```

3. **Abra o arquivo principal**
